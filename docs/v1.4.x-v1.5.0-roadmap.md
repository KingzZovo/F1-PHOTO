- ✅ done — committed on `main` and 6-point sign-off (fmt / clippy / test --lib / release build + repack / smoke as f1u / proactive log diagnosis) is green.
- 🟡 next — accepted as the immediate follow-on after Step C; not yet started.
- ⏸ deferred / optional — explicitly parked per King's direction or contingent on dataset / business decisions.
